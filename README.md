Resume
=============

[简历地址](http://resume.yannan.me/)

My online resume, Build with Node and served by GitHub Pages.

Usage:

**clone代码到本地**

		git clone https://github.com/Southswallows/resume

**本机安装Node**
		
		pip install node

**本机全局安装gulp**

		npm install -g gulp
**resume文件夹内安装gulp、gade**
		
		npm install --save-dev gulp
		npm install gade
**修改`data.js`**

**编译**
 
		`gulp`
**生成了index.html文件**
